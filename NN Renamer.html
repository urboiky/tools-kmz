Rename semua placemark di file KMZ secara berurutan

Tool ini mengganti semua nama placemark pada file KML di dalam KMZ secara berurutan dari placemark paling atas ke bawah. File foto/evidence di dalam KMZ tetap dipertahankan karena seluruh isi arsip selain file KML dibiarkan apa adanya.

Pengaturan

Download hasil KMZ
Siap. Upload file KMZ lalu klik Proses KMZ.
Belum ada proses.

Info

Pola rename yang dihasilkan:
  • placemark paling atas → ?-1
  • placemark berikutnya → ?-2
  • ...
  • placemark berikutnya → ?-100

Catatan penting:
  • Foto/evidence di dalam KMZ tidak dihapus.
  • Yang diubah hanya teks <name> pada tiap Placemark, tanpa filter nama lama.
  • Tool akan mencari file .kml di dalam KMZ secara otomatis.
  • Jika file KMZ punya lebih dari satu KML, semua KML akan dicek.
